## ğŸš€ How to Run Locally

Since this repository contains multiple projects, you need to navigate into the specific project folder you want to run.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/amanm2004/YOUR_REPO_NAME.git](https://github.com/amanm2004/YOUR_REPO_NAME.git)
    ```

2.  **Navigate to the project folder:**
    Choose the project you want to test (e.g., `backgroundChanger`).
    ```bash
    cd backgroundChanger
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Run the app:**
    ```bash
    # For Android
    npm run android

    # For iOS (Mac only)
    npm run ios
    ```
